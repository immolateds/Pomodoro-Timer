# Pomodoro Timer

A cozy pomodoro timer built with HTML, CSS, and JavaScript.

## Features
- 25 minute focus sessions with short and long breaks
- Animated progress ring
- Session tracking dots
- Warm/cool color theme that shifts between focus and break modes

## Use it
Open index.html in any browser, or visit the live version: [https://immolateds.github.io/Pomodoro-Timer/]
